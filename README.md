# 鋼鐵前線 Metal Front

原創橫向卷軸射擊小遊戲，玩法方向類似經典街機跑射闖關，但素材、角色、關卡與名稱都是新做的。

## 內容

- 5 個關卡：港口、叢林、沙漠、雪地、核心基地。
- 每關有復活點，走到旗幟後會更新 checkpoint 並自動存檔。
- 可手動存檔、讀取存檔，使用瀏覽器 `localStorage`。
- 可吃武器升級道具：散彈、雷射、火箭，以及補血、護盾、急速射擊、彈藥、加命、榴彈補給。
- 敵人會依關卡規則刷新，並有每關 boss。
- 每關加入可跳上的平台，敵人也會跳躍、閃避和移動射擊。
- 難度採平衡曲線逐關上升，會反映在敵人血量、傷害、射擊頻率和刷新速度，但不再硬拉到第 5 關爆量。
- 新增高台跳下的突擊兵與飛行敵人，後面關卡會更常出現。
- 支援鍵盤與手機觸控，手機建議橫向玩。

## 操作

- A / D 或方向鍵：移動
- S / 下方向鍵：蹲下躲避一般直線射擊
- W / 空白鍵 / 上方向鍵：跳躍
- J：射擊
- K：榴彈
- P：暫停
- 手機：左下移動，右下射擊、蹲下、跳躍、榴彈

## 開啟方式

直接開 `index.html` 通常可以玩。若瀏覽器限制本機檔案，請在此資料夾上一層執行：

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

然後打開：

```text
http://127.0.0.1:4173/metal-front/index.html
```

## 手機建議

iPhone Safari 橫向時，如果上方網址列或分頁列佔掉太多空間，請用分享按鈕把頁面「加入主畫面」，再從主畫面開啟。這個版本有加 fullscreen/landscape PWA 設定，主畫面開啟會比 Safari 分頁內更適合遊玩。

控制鍵已攔截雙擊縮放與 Safari 手勢縮放。如果瀏覽器仍因分頁列影響高度，加入主畫面開啟會最穩。
