# 鋼鐵前線 Metal Front

原創橫向卷軸射擊小遊戲，玩法方向類似經典街機跑射闖關，但素材、角色、關卡與名稱都是新做的。

## 內容

- 5 個關卡：港口、叢林、沙漠、雪地、核心基地。
- 每關有復活點，走到旗幟後會更新 checkpoint 並自動存檔。
- 可手動存檔、讀取存檔，使用瀏覽器 `localStorage`。
- 可吃武器升級道具：散彈、雷射、火箭，以及補血、榴彈補給。
- 敵人會依關卡規則刷新，並有每關 boss。
- 支援鍵盤與手機觸控，手機建議橫向玩。

## 操作

- A / D 或方向鍵：移動
- W / 空白鍵 / 上方向鍵：跳躍
- J：射擊
- K：榴彈
- P：暫停
- 手機：左下移動，右下射擊、跳躍、榴彈

## 開啟方式

直接開 `index.html` 通常可以玩。若瀏覽器限制本機檔案，請在此資料夾上一層執行：

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

然後打開：

```text
http://127.0.0.1:4173/metal-front/index.html
```
